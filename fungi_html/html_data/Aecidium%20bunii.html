Nomenclature data for Aecidium bunii

Puccinia bulbocastani Fuckel 1870 (Urediniomycetes, Uredinales)

= Aecidium bulbocastani Cumino 1805

= Aecidium bunii DC. 1806

    ≡ Caeoma buniatum Link 1825  Note: Superfluous.

= Puccinia bunii G. Winter 1884  Note: Introduction of new name based on telia (ICBN Art. 59).

Distribution: Widespread.

Disease Note: Rust.

Host: Apiaceae.

Supporting Literature:
