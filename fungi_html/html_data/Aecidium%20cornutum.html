Nomenclature data for Aecidium cornutum

Gymnosporangium cornutum Arthur ex F. Kern 1911 (Urediniomycetes, Uredinales)

= Aecidium cornutum J.F. Gmel : Pers. 1792  Note: From Roestelia cornuta - Persoon (1801) sanctioned the name, Aecidium cornutum, while listing Lycoperdon corniferum and Lycoperdon corniculatum as synonyms. Fries (1849) seemingly based his combination on Aecidium cornutum, but incorrectly attributed that epithet to Ehrenb., the author of Lycoperdon corniculatum.

    ≡ Roestelia cornuta (J.F. Gmel : Pers.) Fr. 1849

     Ceratitium cornutum (J.F. Gmel : Pers.) Rabenh. 1851  Note: Provisional genus. Nom. inval.

    ≡ Gymnosporangium cornutum (J.F. Gmel : Pers.) Arthur 1909

     Ceratitium cornutum (J.F. Gmel : Pers.) Rabenh. 1851  Note: Provisional genus. Nom. inval.

    ≡ Caeoma cornuta (J.F. Gmel : Pers.) Schltdl. 1824

     Ceratitium cornutum (J.F. Gmel : Pers.) Rabenh. 1851  Note: Provisional genus. Nom. inval.

= Podisoma juniperinum Oerst. 1866

Notes: This name is an illegitimate later homonym of Gymnosporangium cornutum Arthur 1909 which is based on an anamorphic type. Kern attributed this name to Arthur when he validated the species based on a teleomorphic type. No obvious legitimate name exists for this species. Gymnosporangium juniperinum (L.) Fr. 1832 based on Tremella juniperina L. 1753., Gymnosporangium juniperi Link 1825, and Gymnosporangium aurantiacum Chevall. 1826 are potential synonyms, however, they have been used variously by different authors including Arthur (1909, 1912), Kern (1911, 1973), and Wilson and Hendersen (1966). Due to the ambiguous nature of these latter names, their use should probably be abandoned and the name Gymnosporangium cornutum should be conserved by one of several options in the ICBN to preserve current use.

Distribution: Temperate northern hemisphere.

Disease Note: Rust.

Host: Telial state on Juniperus spp. (Cupressaceae), aecial state on several genera of Rosaceae.
