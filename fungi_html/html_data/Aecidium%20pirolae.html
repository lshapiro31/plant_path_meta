     Chrysomyxa pirolata G. Winter 1884

= Chrysomyxa ramischiae Lagerh. 1909

Notes: Anamorph in a teleomorph genus (Art. 59.6); Rostrup did not provide a description of the teleomorphic state. From Aecidium pyrolae - Aecidium pyrolae is the original spelling. This name has frequently been cited as pirolae, but it is based on the host genus Pyrola. With the change to unit nomenclature, the name of this rust fungus should be Chrysomyxa pyrolae.

Distribution: Northern hemisphere.

Substrate: On living woody plant parts.

Disease Note: Heteroecious rust.

Host: Spermogonia and aecia on Picea (Pinaceae); uredinia and telia on Pyrola (Ericaceae).
