Nomenclature data for Agaricus atricapillus

Pluteus cervinus (Schaeff.) P. Kumm. 1871 (Basidiomycetes, Agaricales)

≡ Agaricus cervinus Schaeff. 1774

= Agaricus atricapillus Batsch 1786

    ≡ Pluteus atricapillus (Batsch) Fayod 1889

= Pluteus cervinus var. griseoviridis Henn. 1904

  Variant spelling Pluteus cervinus var. griseo-viridis Henn. 1904

Notes: For numerous additional synonyms, see Index Fungorum.

Distribution: Europe, North America.

Substrate: Wood, sawdust, etc.

Disease Note: Saprobic; not a pathogen.
