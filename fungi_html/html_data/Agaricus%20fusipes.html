Nomenclature data for Agaricus fusipes

Gymnopus fusipes (Bull. : Fr.) Gray 1821 (Basidiomycetes, Agaricales)

≡ Agaricus fusipes Bull. : Fr. 1783 [1782-83]

≡Collybia fusipes (Bull. : Fr.) Quél. 1872

≡Rhodocollybia fusipes (Bull. : Fr.) Romagn. 1978

Notes: For numerous additional synonyms, see Index Fungorum.

Distribution: Europe.

Substrate: Roots, trunks.

Disease Note: Root rot, crown dieback.

Host: Quercus spp. (Fagaceae).

Supporting Literature:
