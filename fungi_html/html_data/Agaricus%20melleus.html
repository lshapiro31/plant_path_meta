Nomenclature data for Agaricus melleus

Armillaria mellea (Vahl : Fr.) P. Kumm. 1871 (Basidiomycetes, Agaricales)

≡ Agaricus melleus Vahl : Fr. 1790

≡Armillariella mellea (Vahl : Fr.) P. Karst. 1881

≡Clitocybe mellea (Vahl : Fr.) Ricken 1915

≡Lepiota mellea (Vahl : Fr.) J.E. Lange 1915

= Rhizomorpha fragilis Roth ex Pollini 1791

= Rhizomorpha subcorticalis Pers. ex Gray 1821

Notes: A wide number of possible taxa and/or ecotypes have been included under this name. Many of these entities utilize different ecological niches, including wood and litter decomposition, primary and secondary root pathogens, mycorrhiza formation, and mycoparle epithets possibly belonging to Armillaria mellea. Some outdated nomenclature and names relating to the so called anamorph are still linked, but should not be used. From Rhizomorpha subcorticalis - Boerema and Verhoeven (Neth. J. Pl. Pathol. 78(1):5. 1972).

Distribution: Cosmopolitan.

Substrate: Roots, collar, butt.

Disease Note: Rots; an aggressive pathogen.

Host: Numerous trees, shrubs and some herbaceous plants.
