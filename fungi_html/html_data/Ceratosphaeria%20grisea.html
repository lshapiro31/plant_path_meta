Nomenclature data for Ceratosphaeria grisea

Pyricularia grisea Sacc. 1880 (Ascomycetes, Incertae sedis)

    Variant spelling Piricularia grisea Sacc. 1880

≡Dactylaria grisea (Sacc.) Shirai 1910

= Fusicladium dubiosum Speg. 1886

    ≡ Pyricularia dubiosa (Speg.) Viégas 1946

= Ceratosphaeria grisea T.T. Herbert 1971

    ≡ Magnaporthe grisea (T.T. Hebert) M.E. Barr 1977

    ≡ Phragmoporthe grisea (T.T. Hebert) M. Monod 1983

= Trichothecium griseum Cooke 1882  Note: This name is a nom. nud. There is no clear reference to Saccardo's name in 1881 and the acceptance of a reference is ambiguous, but some list this name as the validly published combination Trichothecium griseum (Sacc.) Cooke. This would be correct depending on one's interpretation.

= Trichothecium griseum Cooke 1882  Note: This name is a nom. nud. There is no clear reference to Saccardo's name in 1881 and the acceptance of a reference is ambiguous, but some list this name as the validly published combination Trichothecium griseum (Sacc.) Cooke. This would be correct depending on one's interpretation.

Distribution: Widespread.

Substrate: On living leaves.

Disease Note: Leaf spot, leaf blight.

Host: Digitaria (Poaceae); report from Costus spiralis (Costaceae).
