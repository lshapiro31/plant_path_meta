Notes: Anamorph thielaviopsis-like. From Ceratocystis filiformis - This name is not valid as no type was indicated.

Distribution: Widespread, Africa, Asia, North America, etc.

Substrate: Wood.

Disease Note: Sapstain.

Host: Multiple genera in multiple families.
