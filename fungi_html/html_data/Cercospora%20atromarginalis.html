Nomenclature data for Cercospora atromarginalis

Pseudocercospora atromarginalis (G.F. Atk.) Deighton 1976 (Ascomycetes, Mycosphaerellales)

    Variant spelling Pseudocercospora atro-marginalis (G.F. Atk.) Deighton 1976

≡ Cercospora atromarginalis G.F. Atk. 1892  Note: 'atramarginalis' is another variant.

      Variant spelling Cercospora atro-marginalis G.F. Atk. 1892

= Cercospora nigri Tharp 1917

= Cercospora rigospora G.F. Atk. 1892

= Cercospora solani-nigri Chidd. 1962  Note: As 'soloni-nigri'. The genus misspelled as Cercosposa.

= Cercospora tosensis Henn. 1904

Distribution: Cosmopolitan.

Substrate: Leaves.

Disease Note: Leaf spot.

Host: Solanum spp. (Solanaceae) and a few other questionable reports.
