  Variant spelling Uredo elephantopodis Schwein. 1822  Note: Original spelling. Genitive of elephantopus corrected to elephantopi.

    ≡ Coleosporium elephantopi (Schwein.) Thüm. 1878  Note: Anamorph in a teleomorphic genus (Art. 59.6); this name was published as a combination based on an anamorph, without description of the teleomorphic state.

Notes: Oldest epithet not in correct genus. From Tubercularia carnea - This is a revalidated name, published previous to 1821. This rust was misclassified by Bosc as an ascomycete. According to Arthur (1906), the type is not extant but Bosc's description and illustration is sufficient. From Uredo elephantopi - This is the earliest name for the uredinial state of this rust. Not Uredo elephantopi Henn. 1896 (as elephantopodis), a taxonomic synonym described as a new species from Brazil; nor Uredo elephantopi Petch 1912 (as elephantopodis) from Sri Lanka.

Distribution: North America, South America, Asia (China, Himalaya).

Substrate: Leaves.

Disease Note: Heteroecious rust.

Host: Aecial host - Pinus spp. (Pinaceae); telial host - Elephantopus and Vernonia spp. (Asteraceae).
