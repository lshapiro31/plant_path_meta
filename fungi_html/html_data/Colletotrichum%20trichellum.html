Nomenclature data for Colletotrichum trichellum

Colletotrichum trichellum (Fr. : Fr.) Duke 1928 (Ascomycetes, Phyllachorales)

≡ Sphaeria trichella Fr. : Fr. 1823

≡Vermicularia trichella (Fr. : Fr.) Fr. 1828

≡Amerosporium trichellum (Fr. : Fr.) Lind 1913

≡Exosporium trichellum (Fr. : Fr.) Link 1825

= Colletotrichum gloeosporioides var. hederae Pass. 1889

= Colletotrichum hedericola Laubert 1907

= Cheilaria helicis Desm. 1847

    ≡ Colletotrichum helicis (Desm.) Morgan-Jones 1971

    ≡ Monostichella helicis (Desm.) Höhn. 1916

    ≡ Gloeosporium helicis (Desm.) Oudem. 1877

Distribution: Widespread.

Substrate: Living leaves.

Disease Note: Anthracnose.

Host: Hedera spp. (Araliaceae) and other reported hosts.
