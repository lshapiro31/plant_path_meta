Nomenclature data for Cronartium filamentosum

Cronartium coleosporioides Arthur 1907 (Urediniomycetes, Uredinales)

= Peridermium stalactiforme Arthur & F. Kern 1906

Notes: Oldest epithet not in the correct genus.

Distribution: Western North America.

Substrate: Branches.

Disease Note: Stalactiform blister rust.

Host: Heteroecious; aecial host - Pinus spp. (Pinaceae); telial host Castilleja spp. (Orobanchaceae).
