Nomenclature data for Cronartium strobilinum

Cronartium strobilinum Hedgc. & Hahn 1922 (Urediniomycetes, Uredinales)

= Caeoma strobilinum Arthur 1906  Note: From Peridermium strobilinum - Southern cone rust. Heteroecious; aecial host - Pinus spp.; telial host - Quercus spp.

    ≡ Peridermium strobilinum (Arthur) R.S. Peterson 1967

Notes: Southern cone rust. Heteroecious; aecial host - Pinus spp.; telial host - Quercus spp.

Distribution: Southeastern United States.

Substrate: On living leaves and pine cones.

Disease Note: Southern cone rust, causes hypertrophy.

Supporting Literature:

Peterson, R.S. 1967. The Peridermium species on pine stems. Bull. Torrey Bot. Club 94: 511-542.
