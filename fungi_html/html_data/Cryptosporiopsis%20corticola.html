Nomenclature data for Cryptosporiopsis corticola

Pezicula corticola (C.A. Jorgensen) Nannf. 1932 (Ascomycetes, Helotiales)

≡ Neofabraea corticola C.A. Jørg. 1930

= Dermatea corticola G. Arnaud 1923  Note: Questionable synonym.

= Myxosporium corticola Edgerton 1908

  ≡Cryptosporiopsis corticola (Edgerton) Nannf. 1932

Distribution: Temperate regions.

Substrate: On bark.

Disease Note: Bark canker.

Host: On Rosaceae.
