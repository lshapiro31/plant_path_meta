Nomenclature data for Dothiora pyrenophora

Dothiora pyrenophora (Fr. : Fr.) Fr. 1849 (Ascomycetes, Dothideales)

≡ Dothidea pyrenophora Fr. : Fr. 1818

= Dothichiza sorbi Lib. 1884

= Micropera sorbi Thüm. 1880

Distribution: Temperate North America; Europe.

Substrate: Branches.

Disease Note: Saprobic.

Host: Sorbus (Rosaceae).
