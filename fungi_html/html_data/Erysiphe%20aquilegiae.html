Nomenclature data for Erysiphe aquilegiae

Erysiphe aquilegiae DC. 1815 (Ascomycetes, Erysiphales)

≡Ischnochaeta aquilegiae (DC.) Sawada 1959

Notes: Includes the type variety.

Distribution: Cosmopolitan.

Substrate: Living leaves, stems, petioles.

Disease Note: Powdery mildew.

Host: Ranunculaceae; report from Cataranthus roseus (Apocynaceae).
