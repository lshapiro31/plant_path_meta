= Cercospora therryana Roum. 1882

Notes: This nomenclator follows Crous and Braun (2003). Several elements were not independently verified. Includes type variety.

Distribution: Northern hemisphere.

Substrate: Leaves.

Disease Note: Leaf spot.

Host: Several taxa in Onagraceae especially Epilobium spp.
