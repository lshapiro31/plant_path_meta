Nomenclature data for Helminthosporium melastomacearum

Spiropes dorycarpus (Mont.) M.B. Ellis 1968 (Ascomycetes, Incertae sedis)

= Helminthosporium conspicuum McAlpine 1897

= Podosporium densum Pat. 1897

= Helminthosporium melastomacearum F. Stevens 1918

= Napicladium myrtacearum Speg. 1888

    ≡ Sporhelminthium myrtacearum (Speg.) Speg. 1918

= Helminthosporium orbiculare Lév. 1846

= Helminthosporium panici F. Stevens 1918

= Helminthosporium parathesicola F. Stevens 1918  Note: As 'parathesicolum'.

Distribution: Widespread.

Disease Note: Hyperparasite of fungi.

Host: Several genera in several families.

Supporting Literature:

Ellis, M.B. 1968. Dematiaceous Hyphomycetes IX. Mycol. Pap. 114: 1-44.
