Nomenclature data for Hypoxylon cubense

Xylaria cubensis (Mont.) Fr. 1851 (Ascomycetes, Xylariales)

≡ Hypoxylon cubense Mont. 1840

≡Xylaria papyrifera subsp. cubensis (Mont.) D. Hawksw. 1973

≡Xylosphaera papyrifera subsp. cubensis (Mont.) Dennis 1961

= Xylaria corniformis (Fr.) Fr. 1849

= Sphaeria flabelliformis Schwein. : Fr. 1825  Note: From Xylaria flabelliformis - Anamorph in a teleomorphic genus (Rogers 1984). From Xylocoremium flabelliforme - This is the type of the genus Xylocoremium.

    ≡ Xylocoremium flabelliforme (Schwein. : Fr.) J.D. Rogers 1984

    ≡ Xylaria flabelliformis (Schwein. : Fr.) Berk. & M.A. Curtis 1868

    ≡ Isaria flabelliformis (Schwein. : Fr.) Lloyd 1916

= Thelephora liliputiana Mont. 1840

= Merisma nigripes Schwein. 1822

= Thelephora rosella Peck 1884

Notes: Xylaria fusca Lloyd 1918 is a possible synonym (Rogers 1984). Dennis 1961 lists possible additional synonyms. From Sphaeria flabelliformis - Replacement name for Merisma nigripes; sanctioned by Fries From Thelephora liliputiana - Not Speg. 1889. From Xylaria corniformis - Anamorph in a teleomorphic genus (Rogers 1984).This name has been frequently misapplied (see Rogers 1984). Oldest epithet not in the correct genus.

Distribution: Cosmopolitan.

Substrate: Dead wood.

Disease Note: White rot.

Host: Woody angiosperms.
