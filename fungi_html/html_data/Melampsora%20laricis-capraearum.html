Nomenclature data for Melampsora laricis-capraearum

Melampsora caprearum Thüm. 1879 (Urediniomycetes, Uredinales)

    Variant spelling Melampsora capraearum Thüm. 1879  Note: Orthographical specialists state that this should be spelled without the extra 'a' due to the spelling of the host Salix caprea.

= Melampsora farinosa J. Schröt. 1887

= Uredo farinosa Pers. : Pers. 1801

= Melampsora laricis-capraearum Kleb. 1897

  Variant spelling Melampsora larici-capraearum Kleb. 1879

Notes: Thümen (1879, referenced in Wilson and Henderson 1966) includes a large number of older synonyms. In light of the taxonomic confusion of this group, these names are not included at the moment. From Uredo farinosa - Not Henn. 1897. Oldest epithet not in correct genus. From Melampsora laricis-capraearum - Orthographical specialists state that this should be spelled without the extra 'a' due to the spelling of the host Salix caprea.

Distribution: Asia, Europe.

Disease Note: Rust on larch-willow.

Host: Larix spp. (Pinaceae) for aecial state, Salix spp. (Salicaceae) for uredinial and telial states.
