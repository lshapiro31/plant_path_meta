Nomenclature data for Microsphaera tortilis

Erysiphe tortilis (Wallr. : Fr.) Link 1824 (Ascomycetes, Erysiphales)

≡ Alphitomorpha tortilis Wallr. : Fr. 1819

≡Microsphaera tortilis (Wallr. : Fr.) Speer 1979 [1978]

≡Tigria tortilis (Wallr. : Fr.) Trevis. 1853

≡Trichocladia tortilis (Wallr. : Fr.) Neger 1905

= Erysiphe corni Duby 1830

Notes: Sanctioned by Fries, Syst. Mycol. 3:243, 1829.

Distribution: Eurasia, North America (USA: MO).

Substrate: Powdery mildew.

Disease Note: Powdery mildew.

Host: Cornus spp. (main host - C. sanguinea) (Cornaceae).
