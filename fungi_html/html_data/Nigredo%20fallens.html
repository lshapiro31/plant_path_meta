Nomenclature data for Nigredo fallens

Uromyces trifolii-repentis var. fallens (Arthur) Cummins 1977 (Urediniomycetes, Uredinales)

≡ Nigredo fallens Arthur 1912

≡Uromyces fallens (Arthur) Barthol. 1928  Note: Not Uromyces fallens (Desm.) Kern 1911, name invalid; refers to Uredo fallens Desm. description of anamorph; without description (Art. 41.3).

≡Uromyces trifolii var. fallens (Arthur) Arthur 1934

Notes: Taxonomy controversial, often referred to as Uromyces fallens (Desm.) Kern (nom. inval.) (e.g., Hiratsuka 1992); see Walker 1978 for discussion. Currently used for the macrocyclic autoecious rust on red clover, Trifolium pratense. From Uromyces trifolii-repentis var. trifolii-repentis - The name U. trifolii has been misapplied to this taxon for many years. Taxonomy controversial; commonly misidentified. Sensu Cummins 1978 includes only the macrocyclic autoecious rust of white clover (T. hybridum, T. repens).. See Walker 1978, Laundon 1975 for discussion of nomenclature of related rusts on Trifolium spp. From Uredo fabae var. trifolii - Should be synanamorph but not supported by database structure. Has been used to include Uromyces trifolii-repentis var. fallens (Arthur) Cummins.

Distribution: Cosmopolitan.

Substrate: Living leaves.

Disease Note: Macrocyclic autoecious rust.

Host: Trifolium spp., primarily T. pratense (Fabaceae).
