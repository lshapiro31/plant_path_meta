Nomenclature data for Nimbya scirpicola

Alternaria scirpicola (Fuckel) Sivan. 1984 (Ascomycetes, Pleosporales)

≡ Sporidesmium scirpicola Fuckel 1863

≡Cercospora scirpicola (Fuckel) Zind.-Bakker 1940

≡Clasterosporium scirpicola (Fuckel) Sacc. 1886

≡Nimbya scirpicola (Fuckel) E.G. Simmons 1989

= Sphaeria scirpi Rabenh. 1844

    ≡ Pyrenophora scirpi (Rabenh.) Wehm. 1961  Note: Nomen illeg. Superfluous.

    ≡ Pleospora scirpi (Rabenh.) Ces. & De Not. 1863  Note: Nomen illeg. Superfluous.

= Sphaeria scirpicola DC. : Fr. 1805

    ≡ Macrospora scirpicola (DC. : Fr.) Fuckel 1870

    ≡ Pleospora scirpicola (DC. : Fr.) P. Karst. 1873

    ≡ Pyrenophora scirpicola (DC. : Fr.) E. Müll. 1951

Notes: Sphaeria scirpicola DC. predates Sporidesmium scirpicola Fuckel, but a valid combination in Alternaria already existed and has been retained by Woudenberg et. al. (2013).

Distribution: Widespread in northern hemisphere.

Substrate: Stems.

Disease Note: Orange spot.

Host: Several taxa in the families Cyperaceae, Juncaceae, Poaceae.
