Nomenclature data for Penicillium terrestre

Penicillium crustosum Thom 1930 (Ascomycetes, Eurotiales)

[= Penicillium farinosum Novobr. 1974 - illegitimate later homonym, not included in search]

= Penicillium pseudocasei Abe ex G. Sm. 1963

= Penicillium terrestre sensu Raper & Thom 1949

Notes: The description of P. terrestre given by Raper and Thom (1949) differs from that of the original description given by C.N. Jensen (1912); P. terestre C.N. Jensen is not a synonym. Penicillium farinosum Novobr. 1974 is an illegitimate later homonym of P. farinosum (Holmsk.) Biourge (1923).

Distribution: Cosmopolitan.

Substrate: Numerous plant and non-plant substrates; commonly on pome fruit.

Disease Note: Storage rot, mold.

Host: Numerous hosts.
