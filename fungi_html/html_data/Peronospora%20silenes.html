Nomenclature data for Peronospora silenes

Peronospora silenes G.W. Wilson 1914 (Oomycetes, Peronosporales)

≡Peronospora arenariae var. macrospora Farl. 1884

     Peronospora arenariae-macrospora Farl. 1884  Note: The citation of this name at the species rank with a hyphen is an error for var. macrospora.

Notes: Nom. nov.

Distribution: Widespread.

Disease Note: Downy mildew.

Host: Silene (Caryophyllaceae).

Supporting Literature:

Constantinescu, O. 1991. An annotated list of Peronospora names. Thunbergia 15: 1-110.
