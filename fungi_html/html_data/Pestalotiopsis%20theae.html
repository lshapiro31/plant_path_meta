Nomenclature data for Pestalotiopsis theae

Pseudopestalotiopsis theae (Sawada) Maharachch., K.D. Hyde & Crous 2014 (, )

≡ Pestalotia theae Sawada 1915

      Variant spelling Pestalozzia theae Sawada

≡Pestalotiopsis theae (Sawada) Steyaert 1949

Distribution: Cosmopolitan except North America.

Substrate: Leaves.

Disease Note: Gray blight, leaf spots.

Host: Multiple genera in multiple families.
