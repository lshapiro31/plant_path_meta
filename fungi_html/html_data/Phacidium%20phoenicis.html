Nomenclature data for Phacidium phoenicis

Graphiola phoenicis (Moug. : Fr.) Poit. 1824 (Ustilaginomycetes, Exobasidiales)

≡ Phacidium phoenicis Moug. : Fr. 1823

≡Roestelia phoenicis (Moug. : Fr.) Bonord. 1851

≡Trichodesmium phoenicis (Moug. : Fr.) Chevall. 1826

Notes: It is uncertain if Graphiola phoenicis (Moug. : Fr.) Poit. and Phacidium phoenicis Moug. : Fr. are based on the same type; if not, the latter name would be a taxonomic synonym.

Distribution: Widespread in tropical and subtropical regions.

Substrate: Leaves, rachis.

Disease Note: Leaf spot, yellowing, leaf death.

Host: Phoenix spp. (Arecaceae).
