Nomenclature data for Phacidium pinastri

Tryblidiopsis pinastri (Pers. : Fr.) P. Karst. 1871 (Ascomycetes, Rhytismatales)

≡ Peziza pinastri Pers. : Fr. 1800

≡Cenangium pinastri (Pers. : Fr.) Fr. 1822

≡Phacidium pinastri (Pers. : Fr. ) Fr. 1828

≡Triblidium pinastri (Pers. : Fr. ) Fr. 1849

  Variant spelling Tryblidium pinastri (Pers. : Fr. ) Fr. 1849

≡Tryblidium pinastri (Pers. : Fr. ) Fr. 1849

≡Tympanis pinastri (Pers. : Fr.) Tul. & C. Tul. 1865

= Tryblidiopsis piceae Velen. 1934

= Tryblidiopycnis pinastri Höhn. 1918

Distribution: Circumboreal.

Substrate: On dead and dying branches of conifers, especially common on Picea spp.

Disease Note: Saprobic; associated with dieback but no direct evidence of pathogenicity.

Host: Pinaceae.
