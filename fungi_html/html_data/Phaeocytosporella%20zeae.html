Nomenclature data for Phaeocytosporella zeae

Phaeocytostroma ambiguum (Mont.) Petr. 1927 (Ascomycetes, Incertae sedis)

≡ Sphaeropsis ambigua Mont. 1849

= Phaeocytostroma istrica Petr. 1921

= Phaeocytosporella zeae G.L. Stout 1930

Distribution: Widespread.

Substrate: Roots, stem.

Disease Note: Rot.

Host: Zea mays (Poaceae).
