Nomenclature data for Phoma berberidis

Cucurbitaria berberidis (Pers. : Fr.) Gray 1821 (Ascomycetes, Pleosporales)

≡ Sphaeria berberidis Pers. : Fr. 1794  Note: Not Wahlenb. 1812.

≡Gibberidea berberidis (Pers. : Fr.) Rabenh. ex Kuntze 1898

≡Hypoxylon berberidis (Pers. : Fr.) J. Kickx f. 1841

= Phoma berberidis Sacc. 1878

    ≡ Pyrenochaeta berberidis (Sacc.) Brunaud 1886

= Crotonocarpia moriformis Fuckel 1870

    ≡ Cucurbitaria moriformis (Fuckel) M.E. Barr 1987

Distribution: Temperate northern hemisphere.

Substrate: On limbs and twigs.

Disease Note: Canker.

Host: Primarily on Berberis (Berberidaceae).
