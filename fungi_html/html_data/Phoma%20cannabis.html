= Diplodina parietaria f. cannabina Höhn. 1910

  Variant spelling Diplodina parietariae f. cannabina Höhn. 1910

Notes: Per Chen et al. (2015), the synomomy of Didymella cannabis is unproven. If synonymous, that name provides the oldest epithet.

Distribution: Europe, Asia, North America.

Substrate: Leaves, stems.

Disease Note: Brown-black leaf spot, stem spot.

Host: Cannabis sativa (Cannabaceae).
