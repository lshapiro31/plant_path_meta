Nomenclature data for Phoma cinerascens

Diaporthe cinerascens Sacc. 1882 (Ascomycetes, Diaporthales)

= Phoma cinerascens Sacc. 1879

    ≡ Phomopsis cinerascens (Sacc.) Traverso 1906

  Variant spelling Phomopsis cinerescens (Sacc.) Traverso 1906

  Variant spelling Phoma cinerescens Sacc. 1879

Notes: Oldest epithet not in correct genus.

Distribution: Worldwide with host genus.

Substrate: Branches, stems.

Disease Note: Canker, dieback of weakened/stressed trees.

Host: Ficus spp. (Moraceae).
