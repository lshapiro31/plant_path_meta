Nomenclature data for Phoma telephii

Boeremia telephii (Vestergr.) Aveskamp, Gruyter & Verkley 2010 (, )

≡ Ascochyta telephii Vestergr. 1897

≡Phoma telephii (Vestergr.) Kesteren 1972

= Ascochyta sedi-purpurei Rothers 1929

= Phoma tabifica Kesteren 1971  Note: Not (Prill.) Sacc. 1892, invalid name (erroneous citation).

Distribution: Europe.

Substrate: Stems, leaves.

Disease Note: Purple blotch disease.

Host: Sedum spp. (Crassulaceae).
