Nomenclature data for Phragmidium cylindricum

Phragmidium bulbosum (F. Strauss) Schltdl. 1824 (Urediniomycetes, Uredinales)

≡ Uredo bulbosa F. Strauss 1810

≡Aregma phragmidium subsp. bulbosum (F. Strauss) Fr. 1832

= Phragmidium cylindricum Bonord. 1860

= Puccinia mucronata ß rubi Pers. : Pers. 1801

    ≡ Phragmidium rubi (Pers. : Pers.) G. Winter 1884

Notes: If Strauss did not see teliospores, the nomenclature of this taxon is incorrect. Petrova and Denchev (2004), however, state that Strauss saw the telia.

Distribution: Africa, Asia, Europe. Reports using P. rubi uncertain.

Substrate: Leaves.

Disease Note: Rust.

Host: Rubus spp. (Rosaceae).

Supporting Literature:
