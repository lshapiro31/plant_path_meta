Nomenclature data for Pileolaria terebinthi

Pileolaria terebinthi Castagne 1842 (Urediniomycetes, Uredinales)

≡Uromyces terebinthi (Castagne) G. Winter 1884

= Uredo terebinthi DC. 1815

    ≡ Caeomurus terebinthi (DC.) Kuntze 1898  Note: This genus is sometimes spelled Coeomurus. Uromyces has been conserved against it. This is an anamorph name according to ICBN Art. 59 since there is no description of the telial state.

Notes: This is the type species of Pileolaria. Sydow and Sydow (1910) list some additional archaic synonyms.

Distribution: Africa, Asia, Europe.

Substrate: Leaves.

Disease Note: Rust.

Host: Pistacia spp. (Anacardiaceae).
