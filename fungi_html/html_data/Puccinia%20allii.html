Nomenclature data for Puccinia allii

Puccinia allii F. Rudolphi 1829 (Urediniomycetes, Uredinales)

= Xyloma allii DC. 1815

[= Puccinia alliorum Casp. 1856 - illegitimate later homonym, not included in search]

= Uredo alliorum DC. 1815

[= Uredo alliorum Link 1816? - illegitimate later homonym, not included in search]

Notes: Puccinia mixta Fuckel 1863 is no longer considered a synonym (Boerema 1993). From Uredo alliorum - Cited by Sacc. in Sp. II, p.43 ( = Obs. Ord. Plant. Nat. Diss. II, 1916?). Could not locate protologue; erroneous citation? From Uredo alliorum - Not Link 1816(?) ------ Oldest epithet not in correct genus.

Distribution: Cosmopolitan.

Substrate: Leaves.

Disease Note: Autoecious rust.

Host: Allium spp., primarily leeks, Allium porrum (Amaryllidaceae). Jennings considers the rust of chives, Allium schoenoprasum to be a separate species, Puccinia mixta Fuckel.
