Nomenclature data for Puccinia chaetochloae

Puccinia chaetochloae Arthur 1907 (Urediniomycetes, Uredinales)

= Uredo chaetochloae Arthur 1906

= Puccinia maublancii Rangel 1916

Distribution: Widespread.

Substrate: Leaves.

Disease Note: Rust.

Host: Several taxa of Poaceae.
