Nomenclature data for Puccinia vincae

Puccinia vincae (DC.) Berk. 1836 (Urediniomycetes, Uredinales)

≡ Uredo vincae DC. 1815

= Bullaria vincae Arthur 1906

Distribution: Widespread.

Disease Note: Rust.

Host: Typically Vinca spp. (Apocynaceae).

Supporting Literature:
