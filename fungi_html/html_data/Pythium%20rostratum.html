Nomenclature data for Pythium rostratum

Globisporangium rostratum (E.J. Butler) Uzuhashi, Tojo & Kakish. 2010 (, )

≡ Pythium rostratum E.J. Butler 1907

= Pythium diameson Sideris 1932

Distribution: Cosmopolitan.

Substrate: Roots.

Disease Note: Root rot.

Host: Multiple hosts in multiple families.
