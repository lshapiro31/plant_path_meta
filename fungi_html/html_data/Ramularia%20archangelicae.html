Nomenclature data for Ramularia archangelicae

Ramularia archangelicae Lindr. 1902 (Ascomycetes, Mycosphaerellales)

= Ramularia angelicae Höhn. 1903

= Septocylindrium angelicae Katsuki 1953

= Ramularia grantii Dearn. 1929

= Cylindrosporium vaccarianum Sacc. 1917

Distribution: Asia, Europe, North America.

Substrate: Living leaves.

Disease Note: Leaf spot.

Host: Angelica spp. (Apiaceae).
