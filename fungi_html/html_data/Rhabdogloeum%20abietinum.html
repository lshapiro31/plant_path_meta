Nomenclature data for Rhabdogloeum abietinum

Sarcotrochila balsameae (Davis) Korf 1962 (Ascomycetes, Helotiales)

≡ Phacidium balsameae Davis 1922

≡Stegopezizella balsameae (Davis) Syd. 1924

= Rhabdogloeum abietinum Dearn. 1928

= Gloeosporium balsameae Davis 1922

    ≡ Rhabdogloeopsis balsameae (Davis) Petr. 1925

    ≡ Kabatiella balsameae (Davis) Arx 1957

Notes: The two species described by Davis (1922) are not based on the same type specimen, thus the link between these species is questionable.

Distribution: Northern North America.

Substrate: On needles.

Disease Note: Needle blight, needle cast.
