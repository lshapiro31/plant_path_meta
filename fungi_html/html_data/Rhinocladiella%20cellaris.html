Nomenclature data for Rhinocladiella cellaris

Zasmidium cellare (Pers. : Fr.) Fr. 1849 (Ascomycetes, Incertae sedis)

≡ Racodium cellare Pers. : Fr. 1794

      Variant spelling Rhacodium cellare Pers. 1794

≡Cladosporium cellare (Pers. : Fr.) Schanderl 1936

≡Antennaria cellaris (Pers. : Fr. ) Fr. 1829

≡Rhinocladiella cellaris (Pers. : Fr.) M.B. Ellis 1971

Distribution: Europe.

Substrate: Commonly found on brickwork and adjacent timber in wine cellars.

Disease Note: Not pathogenic.
