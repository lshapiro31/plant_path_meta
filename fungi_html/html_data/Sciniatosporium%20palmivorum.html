Nomenclature data for Sciniatosporium palmivorum

Stigmina palmivora (Sacc.) S. Hughes 1952 (Ascomycetes, Dothideales)

≡ Exosporium palmivorum Sacc. 1898

≡Cercospora palmivora (Sacc.) Nannizzi 1928 [1927]

≡Sciniatosporium palmivorum (Sacc.) Morgan-Jones 1971

= Exosporium preisii Bubák 1904

     Cercospora preisii Bubák 1903

Notes: Cercospora preisii is invalid since there seems to be no description in the Rabenhorst exsicattae. Bubák (1904, Annales Mycologici 2:400-401) accepted this fungus in the genus Exosporium when the name was validly published with a description. He stated that Cercospora preisii was used in the exsiccatae, and it seems that this name is not accepted here or proposed as an alternate name. Thus, only the Exosporium name is valid.

Distribution: Widespread, especially in subtropical to tropical regions.

Substrate: Leaves.

Disease Note: Leaf spot.

Host: Several taxa of Arecaceae.
