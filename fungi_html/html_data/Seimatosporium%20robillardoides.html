Nomenclature data for Seimatosporium robillardoides

Bartalinia robillardoides Tassi 1900 (Ascomycetes, Incertae sedis)

≡Seimatosporium robillardoides (Tassi) Arx 1981

= Amphiciliella eriobotryae Höhn. 1919

     Amphiciliella eryobotryae SaccOnly

= Bartalinia nervisequa Tassi 1972

Distribution: Widespread.

Substrate: Dead leaves and litter.

Disease Note: Saprobic.

Host: Various host plants.

Supporting Literature:

Nag Raj, T.R. 1993. Coelomycetous anamorphs with appendage-bearing conidia. Mycologue Publications, Waterloo, Ontario, 1-1101 pages.
