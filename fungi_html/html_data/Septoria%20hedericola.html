Nomenclature data for Septoria hedericola

Septoria hedericola (Fr. : Fr.) Jørst. 1965 (Ascomycetes, Mycosphaerellales)

≡ Sphaeria hedericola Fr. : Fr. 1823

      Variant spelling Sphaeria hederaecola Fr. : Fr. 1823

≡Septoria hederae Desm. 1843  Note: Nom. illeg. via superfluous.

≡Depazea hedericola (Fr. : Fr.) Rabenh. 1844  Note: As 'hederaecola'.

≡Sphaerella hedericola (Fr. : Fr.) Cooke 1874  Note: As 'hederaecola'. This combination was based on Fries' type, which is now considered to represent an anamorph. Cooke observed teleomorph material.

= Asteromella hederacea Petr. 1958 [1957]  Note: Nom. nov. According to Aptroot (2006), this is the spermatial state.

  [≡ Asteromella hederae (Sacc. & Roum.) Petr. 1957 [1956] - illegitimate later homonym, not included in search]  Note: Not C. Mass. 1900. Nom. illeg.

    ≡ Phyllosticta hederae Sacc. & Roum. 1882

= Carlia hedericola Höhn. 1925  Note: From Mycosphaerella hedericola - As '(Desm.) Lindau'.

    ≡ Mycosphaerella hedericola (Höhn.) Lindau 1897

[≡Sphaeria hedericola Desm. 1850 - illegitimate later homonym, not included in search]

= Septoria insularis Berk. & Broome 1854

= Sphaeria lichenoides var. hederaecola DC. 1815

Notes: From Carlia hedericola - As 'hederaecola' and (Desm.) Höhn. This is a nom. nov. based on the type of Desmazier. From Sphaeria hedericola - As 'hederaecola'. Not Fr. : Fr. 1823. Nomen illeg.

Distribution: Asia, Europe.

Substrate: Leaves.

Disease Note: Leaf spot.

Host: Hedera spp. (Araliaceae).
