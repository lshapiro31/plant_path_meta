Nomenclature data for Sphaeria nebulosa

Phomatodes nebulosa (Pers. : Fr.) Qian Chen & L. Cai 2015 (, )

≡ Sphaeria nebulosa Pers. : Fr. 1800 [1799]

≡Exormatostoma nebulosa (Pers. : Fr.) Gray 1821

≡Phoma nebulosa (Pers. : Fr.) Berk. 1860

≡Sphaeropsis nebulosa (Pers. : Fr.) Fr. 1849

= Leptophoma paeoniae Höhn. 1915

Notes: From Sphaeronaema paeoniae - Without description. From Exormatostoma nebulosa - As 'nebulosum'.

Distribution: North America, Europe, Asia (India), Australia, New Zealand.

Substrate: Dead stems, soil-borne.

Disease Note: Saprophyte.

Host: Herbaceous plants especially Urticaceae.
