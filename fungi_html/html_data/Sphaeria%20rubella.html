Nomenclature data for Sphaeria rubella

≡ Sphaeria rubella Pers. : Fr. 1801

≡Ophiobolus rubellus (Pers. : Fr.) Sacc. 1881

Substrate: Leaves.

Disease Note: Leaf spot.
