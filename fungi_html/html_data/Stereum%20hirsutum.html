Nomenclature data for Stereum hirsutum

Stereum hirsutum (Willd. : Fr.) Gray 1938 (Basidiomycetes, Russulales)

≡ Thelephora hirsuta Willd. : Fr. 1821

= Auricularia reflexa Bull. 1786

    ≡ Thelephora reflexa (Bull.) Lam. & DC 1805

    ≡ Stereum reflexum (Bull.) Sacc. 1916

Notes: Synonomy follows Lentz (1955); see that work for numerous additional synonyms. Auricularia reflexa Bull. provides the oldest epithet but is unpriorable due to the sanctioning of T. hirsuta.

Distribution: Widespread.

Substrate: Usually dead angiosperms; occasionally gymnosperms and living trees.

Disease Note: Heart rot, necrosis, associated with esca; usually saprobic.

Host: Numerous hosts.
