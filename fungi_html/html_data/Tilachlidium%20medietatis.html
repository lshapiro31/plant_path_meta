Nomenclature data for Tilachlidium medietatis

Sarocladium strictum (W. Gams) Summerb. 2011 (Ascomycetes, Incertae sedis)

≡ Acremonium strictum W. Gams 1971

= Cephalosporium acremonium Corda 1839

= Tilachlidium medietatis Novobr. 1972

Notes: Older name not in correct genus.

Distribution: Cosmopolitan.

Substrate: Leaves, petioles.

Disease Note: Leaf spot, wilt.

Host: Numerous hosts.
