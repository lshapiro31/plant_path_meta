Nomenclature data for Tremella sabinae

Gymnosporangium sabinae (Dicks.) G. Winter 1884 (Urediniomycetes, Uredinales)

≡ Tremella sabinae Dicks. 1785

= Puccinia juniperi Pers. : Pers. 1794  Note: Sanctioned in 1801.

Notes: Laundon (1975) discussed the nomenclatural problems associated with this fungus. Even though Gymnosporangium fuscum has been adopted by several authors, this name is not correct because DC. cited Puccinia juniperi and other names in synonymy. One of these names should have given the name bringing epithet at that time. Thus, the DC. name is a nomen illeg. Several archaic synonyms including some for the aecial state are known for this species, but they are not included here.

Distribution: Widespread.

Substrate: Living leaves, petioles.

Disease Note: Rust.

Host: Juniperus spp. (Cupressaceae) telial state, Pyrus spp. (Rosaceae) aecial state.
