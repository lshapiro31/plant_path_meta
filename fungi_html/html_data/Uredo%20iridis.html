Nomenclature data for Uredo iridis

Puccinia iridis Wallr. 1844 (Urediniomycetes, Uredinales)

≡Dicaeoma iridis (DC.) Kuntze 1898

≡Trichobasis iridis (Wallr.) Cooke 1875

= Uredo iridis DC. 1808

Distribution: Temperate regions.

Substrate: Leaves.

Disease Note: Heteroecious rust.

Host: Aecial host - Urtica spp. (Urticaceae) and Valeriana spp. (Valerianaceae); telial host - Iridaceae.
