≡ Uredo uredinis Link 1824

= Chrysomyxa albida J.G. Kühn 1883

    ≡ Kuehneola albida (J.G. Kühn) Magnus 1898

Notes: For additional synonyms, see Index Fungorum.

Distribution: Temperate regions.

Substrate: Canes, leaves.

Disease Note: Rust.

Host: Rubus spp. (Rosaceae).
