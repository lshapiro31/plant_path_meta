= Ustilago silenes-nutantis Liro 1924  Note: Liro first validly published this name fide Vanky (2011).

Notes: Sensu lato. From Ustilago violacea var. silenes-nutantis - Nom. inval. From Uredo antherarum a silenes-nutantis - Nom. nud. Nom. inval fide Vanky (2011).

Distribution: Cosmopolitan.

Substrate: Anthers.

Disease Note: Smut.

Host: Caryophyllaceae (subfamily Silenoideae).
