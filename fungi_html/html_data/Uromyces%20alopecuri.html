Nomenclature data for Uromyces alopecuri

Uromyces dactylidis var. poae (Rabenh.) Cummins 1974 (Urediniomycetes, Uredinales)

≡ Uromyces poae Rabenh. 1866

= Uromyces alopecuri C.P. Seym. 1889

= Aecidium ficariae Pers. 1803

= Uromyces ranunculi-festucae Jaap 1905

Notes: Heteroecious; aecial host - Ranunculaceae; telial host - Poaceae.

Distribution: Temperate regions.

Substrate: Living leaves.

Disease Note: Rust fungus.

Host: Poaceae and Ranunculaceae.
