Nomenclature data for Ustilago heufleri

Vankya heufleri (Fuckel) Ershad 2000 (Ustilaginomycetes, Urocystales)

≡ Ustilago heufleri Fuckel 1870

= Ustilago erythronii G.P. Clinton 1873

    ≡ Ustilago ornithogali f. erythronii (G.P. Clinton) De Toni 1888

= Ustilago tulipae G. Winter 1884  Note: Since the combination was based on an invalid name, the description would have validated this as a new species. However, the name is a nom. illeg. via superfluous since U. heufleri was cited as a synonym.

    ≡ Urocystis pompholygodes f. tulipae Rabenh. 1866  Note: Nom. nud. Nom. inval.

Distribution: Widespread in northern hemisphere.

Disease Note: Smut.

Host: Erythronium spp., Tulipa spp. (Liliaceae).
