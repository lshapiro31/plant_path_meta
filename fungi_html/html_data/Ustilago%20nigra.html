Nomenclature data for Ustilago nigra

Ustilago avenae (Pers.) Rostr. 1890 (Ustilaginomycetes, Ustilaginales)

= Ustilago aegilopsidis Picb. 1932

= Ustilago decipiens Liro 1924

= Ustilago festucae Zundel 1933

= Ustilago haynaldiae Becer. 1970

= Ustilago medians Biedenk. 1894

= Ustilago nigra Tapke 1932

= Ustilago perennans Rostr. 1890

Notes: Based on Uredo segetum unranked avenae Pers. 1797. Several homotypic unranked and infraspecific synonyms exist. Vanky (2011) lists these and some more heterotypic synonyms.

Distribution: Cosmopolitan.

Substrate: Ovaries.

Disease Note: Smut.

Host: Aegilops, Arrhenatherum, Avena, Dasypyrum, Hordeum (Poaceae).
